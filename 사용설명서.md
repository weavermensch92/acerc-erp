## 1. 시작하기 — 로그인

1. 인터넷 브라우저(크롬·엣지·사파리)에서 **회사 ERP 주소** 접속
   - 예: `https://acerc.vercel.app`
2. 로그인 화면이 나옵니다.
   - **ID**: `ace6125332` (도메인 자동 부착)
   - **비밀번호**: 사무실에서 받은 비밀번호
3. **[로그인]** 버튼 클릭 → 대시보드 진입

> 💡 **팁** — 한 번 로그인하면 브라우저를 닫기 전까지 계속 유지됩니다.
> 다른 컴퓨터·휴대폰에서도 같은 ID/비번으로 동시 사용 가능합니다.

| 메뉴 | 무슨 화면? |
|---|---|
| **🏠 대시보드** | 이번달 운영 요약 — 건수 / 청구 / 미결제 / 14일 차트 / 거래처 Top5 |
| **📋 폐기물일보** | 일보 목록 — 가장 자주 쓰는 화면 |
| **👥 거래처** | 거래처 등록 / 수정 / 공유링크 발급 |
| **🧾 거래명세표** | 거래처별·기간별 명세표 조회 + 인쇄 + 일괄 발급 |

- **공용 계정** (왼쪽 아래) — 로그아웃은 여기서

## 3. 일보 입력 (기본)

> 📌 **언제 사용?** — 한 건씩 신중하게 입력할 때.
> 여러 건을 빠르게 입력하려면 [§4 빠른 입력](#4-일보-입력-빠른-입력--여러-건-동시) 보세요.

### 단계

1. 사이드바 **폐기물일보** 클릭
2. 우측 상단 **[+ 새 일보 입력]** 클릭
3. 폼이 4 섹션으로 나뉘어 있어요:

#### ① 거래정보
- **일자** — 오늘 날짜 자동, 필요하면 변경
- **구분** — [반입] (배출자→자사) / [반출] (자사→처리장)
- **거래처** — 이름 입력 (예: "구") → 자동완성 후보 클릭
  - 목록에 없으면 **"+ 새 거래처 등록"** 클릭 → 모달에서 사업자번호 등 입력
- **공사현장** — 거래처 선택 후 활성화. 직접 입력하면 새 현장으로 자동 추가됨

#### ② 폐기물정보
- **성상** — "폐목재", "폐콘크리트" 등. 목록에 없으면 새 이름 입력 → 마스터에 자동 추가
- **처리장** — 동일 (자동 추가)
- **차량번호** — 자유 입력 (예: `경북80바1234`)

#### ③ 금액정보
- **청구 타입** 4 가지 중 1개:
  - **중량기준** (기본) — 중량 × 단가 + VAT
  - **정액** — 건당 정액 + VAT
  - **사급** — 자사 작업 (0원)
  - **면세** — 부가세 0
- **총중량 / 공차중량 / 실중량** —
  - 총중량 입력 → 공차중량 (차량 등록 시 자동 채움) → 실중량 자동 계산 (총-공차)
  - 공차중량은 직접 수정 가능
- **단가** — 거래처 또는 성상의 기본단가 자동 채움. 변경 가능
- **운반비** — 별도 운반료가 있을 때만

#### ④ 비고
- 메모 (선택)

### 자동 계산
입력하는 동안 우측 박스에 **공급가액 / 부가세 / 청구금액** 실시간 표시됩니다.
예: 폐목재 490kg × 70원 → 공급가액 ₩34,300 / 부가세 ₩3,430 / **청구금액 ₩37,730**

### 저장
**[저장]** 클릭 → 일보 상세 화면으로 이동.
잘못 입력한 게 있으면 [수정] 버튼으로 다시 고칠 수 있습니다.

## 4. 일보 입력 (빠른 입력 / 여러 건 동시)

> 📌 **언제 사용?** — 하루치 여러 건을 한 번에 입력. 데이터 엔트리 담당자에게 추천.
> 단, **PC 전용** (모바일 X). 청구 타입은 **중량기준만** 지원.

### 단계

1. 폐기물일보 우측 상단 **[빠른 입력]** 클릭
2. 5행 빈 표가 나옵니다. 부족하면 [+ 행 추가] / [+ 5행] / [+ 10행]
3. 각 셀 클릭 → 입력
   - **거래처 / 성상 / 처리장** — 입력 시 자동완성, 없으면 그대로 새로 추가됨
   - 거래처 선택 시 **단가 자동 채움**
4. **Tab** 키로 다음 셀, **Shift+Tab** 으로 이전 셀
5. 하단 합계 박스에 자동 계산 합계 + 정상/오류 카운트
6. **[N건 저장]** → 정상 행만 일괄 등록

### 시각 안내
- **분홍 배경 행** = 필수 정보(거래처/성상/일자) 누락 → 등록 제외
- **회색 배경 행** = 빈 행 → 무시
- **흰 배경 행** = 정상

### 차이 — 기본 입력 vs 빠른 입력
| 항목 | 기본 (`/logs/new`) | 빠른 입력 (`/logs/bulk`) |
|---|---|---|
| 한 번에 등록 | 1건 | 여러 건 |
| 청구 타입 | 4가지 모두 | 중량기준만 |
| 사진 토글 | 가능 | X |
| 모바일 | 가능 | PC 전용 |

## 5. 일보 수정 / 삭제

### 단일 일보 수정

1. 폐기물일보 목록에서 행 클릭 → 상세 화면
2. 우측 액션 패널의 **[수정]** 클릭
3. 모든 필드 변경 가능
4. **[수정 사유]** 입력 (선택, 변경 이력에 기록)
5. **[수정 저장]**

> ⚠️ 거래처가 이미 거래명세표를 받아간 후라면 노란 배너로 안내됩니다.
> 단가·금액 수정 시 거래처에게 카톡으로 변경 안내 권장.

### 단일 일보 삭제

1. 폐기물일보 목록에서 행 클릭 → 상세 화면
2. 검토 대기 상태면 **[반려]** 사용
3. 정식/보관 상태는 [삭제] 직접 버튼은 없고, **다중 선택 삭제** 사용 (아래)

### 여러 일보 한 번에 삭제 (보관 처리)

1. 폐기물일보 목록의 각 행 **체크박스** 클릭 (또는 헤더 체크박스로 전체 선택)
2. 선택하면 상단에 검은 sticky 바 등장: **N건 선택됨**
3. 우측 **[N건 삭제]** 클릭
4. 확인 모달에서 **삭제 사유** 입력 (선택) → **[삭제]**
5. 선택된 일보들이 **보관(archived)** 상태로 변경됨
6. 통계·거래명세표·대시보드에서 자동 제외

### 보관된 일보 보기 / 복원

1. 폐기물일보 목록 상단 상태 칩에서 **[보관(삭제)]** 클릭
2. 회색으로 보관된 일보 목록 표시
3. 행 클릭 → 상세 → [복원] (정식 상태로 되돌림)

> 💡 **삭제는 진짜 삭제가 아닙니다** — DB 에 데이터는 남고, 화면에서만 숨김. 변경 이력에 모두 기록됩니다.

## 6. 검토 승인 워크플로우

### 검토 프로세스 ON / OFF

기본은 **OFF** — 입력하면 바로 정식 등록.
**ON** 으로 바꾸면 — 현장에서 입력된 일보가 사무직원 검토 대기 상태로 들어옵니다.

**바꾸는 법**:
1. 사이드바 **설정** → "검토 프로세스" 섹션
2. 토글 클릭 (회색 → 검은색)
3. 즉시 모든 사용자에게 반영

### 검토 흐름 (ON 일 때)

1. 대시보드 상단에 **검토 대기 N건** 노란 배너 자동 표시
2. 클릭 → 검토 대기 일보 목록 (`/logs?status=pending_review`)
3. 행 클릭 → 일보 상세
4. 사진과 입력값 대조 후:
   - **[승인]** → 정식 등록 → 자동으로 다음 검토 건으로 이동
   - **[반려]** → 사유 입력 → 보관(archived) 처리

## 7. 거래처 관리

### 거래처 등록

1. 사이드바 **거래처** → 우측 상단 **[+ 새 거래처]**
2. 입력 항목:
   - **거래처명** (필수)
   - **사업자번호** (예: `123-45-67890`)
   - **주소 / 담당자 / 연락처**
   - **기본단가** — 일보 입력 시 자동 채움 값 (옵션)
   - **자사 (사급)** 체크 — 자사 작업이면 청구 0원 자동
3. **[등록]** → 상세 페이지로 이동

### 거래처 목록 + 검색

거래처 페이지:
- 검색창에 이름 입력 → [검색]
- 컬럼: 이름 / 사업자번호 / 담당자 / 기본단가 / 거래 건수 / 공유링크 / 자사·외부

### 거래처 상세 + 수정

행 클릭 → 상세 화면:
- 좌측: 거래처 정보 편집 폼 (변경 후 [저장])
- 좌측 하단: 공사현장 목록 (일보 입력 시 자동 추가됨)
- 좌측 하단: 최근 거래 10건
- 우측: 전체 요약 (총 거래 / 청구 / 미결제) + 공유 링크 패널

## 8. 거래처 공유 링크 (셀프 조회)

> 거래처가 인증 없이 자기 거래내역을 직접 볼 수 있는 URL 발급.

### 발급 방법

1. **거래처** → 해당 거래처 클릭 → 상세
2. 우측 **공유 링크** 카드 → **[공유 링크 발급]**
3. 16자리 랜덤 URL 생성됨 (예: `https://acerc.vercel.app/share/abc123def...`)
4. **[링크 복사]** 클릭 → 카톡 / 문자로 거래처에 전달

### 재발급 / 회수

- **🔄 재발급** — 새 토큰 생성, 기존 링크 즉시 무효
- **🗑️ 회수** — 토큰 삭제, 거래처가 더 이상 접근 못 함

### 거래처가 보는 화면

거래처가 받은 링크 클릭 시:
- 자기 거래내역만 표시 (다른 거래처 정보 절대 안 보임)
- 월 선택 (최근 7개월 + 다음달)
- 요약 카드 3 (이번달 건수 / 청구 합계 / 미결제)
- 거래 테이블 (PC) 또는 카드 리스트 (모바일)
- **[거래명세표 인쇄 / PDF 저장]** — 클릭하면 거래명세표 양식이 인쇄/저장됨

## 9. 거래명세표 발급

### 단건 발급 (조회 + 인쇄)

1. 사이드바 **거래명세표**
2. 거래처 + 월 선택 → **[조회]**
3. 화면 상단에 **인라인 편집표** 표시:
   - 단가 / 중량 / 청구금액 / 청구·결제 체크박스 등 직접 수정 가능
   - 변경된 행은 **노란 배경**, 청구금액 색 변경
   - 하단 sticky 바에서 [N건 저장] → DB 즉시 반영
4. 화면 하단에 **거래명세표 양식** 표시 (인쇄용)
5. 우측 상단 **[인쇄 / PDF 저장]** → 브라우저 인쇄 → "PDF 로 저장" 선택

### 일괄 발급 (월말 정기)

> 📌 한 번에 여러 거래처에 명세표 발급 + 거래처별 공유링크 정리.

1. 거래명세표 → 우측 상단 **[+ 일괄 발급]**
2. **기간** 선택 (기본: 이번 달)
3. **거래처 체크박스** — 자동으로 그 월에 거래 있는 거래처만 표시
   - 옆에 거래 건수 + 합계
   - 미결제 있는 거래처는 빨간 점
   - **[전체 선택]** [미결제만] [해제]
4. 비고 입력 (선택)
5. 하단 sticky 바: 선택 합계 확인 → **[N곳 발급]**
6. 결과 화면: 거래처별 카드 (PDF 미리보기 / 공유링크 복사 / 외부 열기)
7. 각 거래처에 카톡으로 공유 링크 보내면 끝

### 발급 이력

거래명세표 우측 상단 **[발급 이력]** → 과거 일괄 발급 기록 확인.

## 10. 처리확인서 / 계량증명서 출력

> 폐기물관리법 표준 양식. 서명란이 비어 있는 PDF.

### 발급 위치

폐기물일보 상세 → 우측 액션 패널 하단 **문서 발급** 섹션:
- **📄 처리확인서** — 배출자/운반자/처리자 3자 정보 + 폐기물 정보 + 서명란 3
- **⚖️ 계량증명서 (3부)** — A4 1장에 배출자용/운반자용/처리자용 3부 분할 (절취선)

## 12. 엑셀 다운로드 / 업로드

### 다운로드 (백업 / 외부 분석용)

1. 폐기물일보 우측 상단 **[엑셀 다운]**
2. 현재 필터 (시작일/종료일/거래처/상태) 적용된 데이터 다운로드
3. 파일명: `종합일보_2026-04-01_2026-04-30.xlsx`
4. Excel / 한컴오피스 / Google Sheets 모두 호환

### 업로드 (대량 일괄 등록)

> 📌 기존 엑셀 파일을 시스템에 일괄 등록할 때.
> ⚠️ **수정이 아닌 추가** — 같은 일보를 다시 올리면 중복됨.

1. 폐기물일보 우측 상단 **[엑셀 업로드]** → `/admin/import` 진입
2. **1. 파일 선택** — `.xlsx` / `.xls` / `.csv` 클릭
3. **2. 헤더 자동 매핑** — 한국어 헤더 자동 인식 (일자/거래처/성상/총중량/공차중량/실중량 등)
4. **3. 검증 통계** — 정상 / 오류 행 분리. 오류 행은 50개까지 표시
5. **4. 일괄 등록** — 정상분만 등록 + 마스터 자동 추가 결과
6. 결과 화면: 등록 N건 / 신규 거래처·성상·처리장·현장 카운트

### 엑셀 헤더 인식 가능한 이름

자동 인식되는 한국어 헤더 (영문도 가능):
- 일자 / 날짜 / date
- 구분 / 반입반출 / direction
- 거래처 / 배출자 / company
- 공사 / 공사명 / 공사현장 / 현장
- 성상 / 폐기물 / 폐기물종류 / waste
- 처리장 / 처리시설 / plant
- 차량 / 차량번호 / vehicle
- 중량 / 실중량 / weight / kg
- 단가 / price
- 운반비 / fee
- 공급가액 / 부가세 / 청구금액 / 합계 / 비고

> 💡 **워크플로우 팁** — 다운로드 → Excel 에서 검토 → 잘못된 행 수정 → 재업로드
> (단, 재업로드 시 중복 추가됨. 기존 행 수정은 시스템 안에서 직접 하세요.)

## 13. 마스터 관리 (성상 / 처리장)

### 진입

사이드바 **마스터** → 두 카드 표시 — 성상 / 처리장

### 성상 (waste_types)

1. **마스터 → 성상**
2. **[+ 새 성상]** → 모달에서 이름 + 기본단가 입력
3. 행의 **✏️** 클릭 → 편집 (이름·단가)
4. 행의 **🗑️** 클릭 → 삭제 (사용 중이면 비활성, 안전장치)

### 처리장 (treatment_plants)

동일한 패턴 — 이름 + 주소.

### 자동 추가

일보 입력 시 새 이름 입력하면 마스터에 자동으로 추가됨. 마스터 페이지는 **사후 정리·이름 통일** 용도.

### 거래처는 별도

거래처는 사이드바 **거래처** 메뉴에서 관리.
공사현장은 거래처 종속 — 거래처 상세 페이지에서 자동 표시.

## 18. 자주 묻는 질문 (FAQ)

### Q1. 일보를 잘못 입력했어요. 어떻게 고치나요?
A. 폐기물일보 → 그 일보 클릭 → 우측 [수정] → 모든 필드 변경 가능.

### Q2. 거래처에서 단가가 틀렸다고 연락 왔어요.
A. 일보 상세 → [수정] → 단가 변경 → 수정 사유 "단가 정정" 입력 → 저장.
거래처가 이미 명세표를 받아간 상태면 노란 배너로 안내됨. 카톡으로 변경 안내 권장.

### Q3. 같은 거래처의 여러 일보를 한번에 결제 처리하고 싶어요.
A. 거래명세표 → 거래처 + 월 선택 → 화면 편집표에서 **결제** 체크박스 일괄 체크 → 하단 [N건 저장].

### Q4. 거래처가 사업자번호 알려달래요.
A. 자사 정보는 **설정 → 자사 정보** 에 입력. 거래처에 보낼 명세표에 자동 포함.

### Q5. 사진을 휴대폰으로 못 찍는데, PC 에서 올릴 수 있나요?
A. 일회용 링크는 휴대폰 친화 디자인이지만 PC 에서도 열 수 있어요. PC 에서 [사진 선택] → 파일 탐색기로 사진 선택.

### Q6. 검토 프로세스를 실수로 켜뒀어요. 끄면 검토 대기 일보는 어떻게 되나요?
A. 검토 대기 일보는 그대로 남아있지만 화면에 안 보입니다. 다시 켜면 즉시 표시. 데이터 손실 X.

### Q7. 거래처가 자기 링크를 잃어버렸어요.
A. 거래처 상세 → 공유 링크 패널 → 그대로 [링크 복사] 후 다시 전달.
완전히 새 링크로 바꾸려면 [🔄 재발급] (기존 링크 즉시 무효).

### Q8. 엑셀에서 수정해서 다시 올리면 기존 데이터가 업데이트되나요?
A. **아니요** — 새 행으로 추가됩니다. 기존 행 수정은 시스템 안에서 직접 하세요 (일보 상세 → [수정] 또는 거래명세표 화면 편집).

### Q9. 보관(삭제) 한 일보를 영구 삭제하고 싶어요.
A. 보안·감사 추적을 위해 영구 삭제는 사무실 → 개발자에게 요청. (직접 SQL 또는 Studio).

### Q10. 자동 백업은?
A. (1) Supabase 가 자동 일일 백업 7일치 보관 (무료). (2) 매일 자정 자체 스냅샷 (cron) 메타 기록.
중요 시점 (분기 마감 등) 에는 [엑셀 다운로드] 로 별도 보관 권장.

## 19. 문제 해결

### 로그인이 안 돼요
1. ID `ace6125332` 정확히 입력했는지 (대소문자)
2. 비밀번호 끝의 `!!` 같은 특수문자 누락 X
3. 그래도 안 되면 사무실 → 비밀번호 재설정 요청

### 화면이 깨져 보여요
- 브라우저 새로고침 (`F5` 또는 `Ctrl+F5`)
- 크롬 / 엣지 / 사파리 최신 버전 권장 (IE 지원 X)
- 모바일은 가로 스크롤 가능

### "검토 대기" 가 안 사라져요
- 설정 → 검토 프로세스 ON 인지 확인
- 일보 상세에서 [승인] 또는 [반려] 처리 후 재로드

### 거래처가 공유 링크에 들어갔는데 데이터가 없대요
- 그 거래처에 그 월 거래가 정말 없는지 확인 (폐기물일보 → 거래처 필터)
- 일보의 status 가 'archived' 면 셀프 페이지에서 안 보임

### 엑셀 업로드 시 매핑 실패
- 헤더 이름 확인 (한국어 또는 영문)
- 첫 행이 헤더, 두번째 행부터 데이터인지 확인
- 빈 행은 자동 제외

### 사진 업로드 시 "링크가 만료" 메시지
- 24시간 경과 = 만료. 새 링크 발급 필요
- "이미 사용됨" 도 동일 (한 번 업로드 후 폐기)
